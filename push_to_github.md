# Scripts para Publicar no GitHub

Execute as seguintes linhas de comando em seu terminal de preferência na raiz do projeto (`c:\Users\46356046830\Documents\Antigravity 2605\06 Game - Megamania remake`) para associar o repositório local ao repositório remoto do GitHub e enviar o código:

```powershell
# 1. Adicionar o repositório remoto do GitHub
git remote add origin https://github.com/jhonattan-prog/Mega_mania.git

# 2. Enviar a branch 'main' para o repositório remoto (definindo a upstream)
git push -u origin main
```

*Nota: Certifique-se de que você já realizou login na sua conta do GitHub no terminal ou que possui as credenciais SSH/HTTPS corretas configuradas.*
